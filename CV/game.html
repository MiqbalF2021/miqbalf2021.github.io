Bermain Game

Alasan saya bermain game karena main game memiliki efek yang baik pada otak dan tubuh. Ini dapat meningkatkan keterampilan kognitif seperti pemecahan masalah, meningkatkan keterampilan sosial melalui bermain dengan teman, dan merangsang responsivitas visual dalam permainan yang memerlukan reaksi cepat. Bermain game juga dapat memberikan hiburan, melepaskan endorfin, dan merangsang kreativitas.

Berikut adalah beberapa game yang saya mainkan saat ini :

  1. Genshin Impact
  2. Mobile Legends
  3. Efootball 2024 Mobile
  4. Chess
